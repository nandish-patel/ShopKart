# ShopKart

## Overview

ShopKart is a simple e-commerce website inspired by Amazon and Flipkart. It allows users to create an account, log in, browse products, add items to a shopping cart, and place an order. The project is built using HTML, CSS, and JavaScript with Local Storage for data persistence.

## Features

* User Signup and Login
* Product Listing
* Add to Cart
* View and Manage Cart
* Checkout Page
* Logout Functionality
* Data stored using Local Storage
* Responsive and user-friendly interface

## Technologies Used

* HTML5
* CSS3 (Single CSS file)
* JavaScript (Embedded inside HTML pages)
* Local Storage

## How to Run

1. Download or clone the project.
2. Open the project folder in Visual Studio Code.
3. Run `index.html` using the Live Server extension, or open it directly in a web browser.
4. Create a new account using the Signup page.
5. Log in and start shopping.
